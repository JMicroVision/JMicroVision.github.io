Coloring Objects

This dialog enables to color all the objects from a look-up table according to the values of the selected descriptor.

 Note: To open this window, select the Object Extraction tool, then the Objects tab and finally click on the Color button.

Sort by: Select an object descriptor.

Ascending: If checked, the objects with the smallest values are colored by the left side of the look-up table and the biggest by the right side.

Histogram: Open the histogram of the selected descriptor.

LUT (look-up table or palette of colors): Select a LUT.

Transfer function: Select Linear, Logarithmic or Exponential. The transfer function sometimes helps to bring out small differences between objects at the beginning or at the end of the range values.

 Note: When closing this window, all the objects retrieve their original color (corresponding to the class which they belongs to).
