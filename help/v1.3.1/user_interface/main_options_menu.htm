Look & Feel

This option allows you to choose a specific design for the JMicroVision interface. The number of look and feel depends on which platform and Java Virtual Machine JMicroVision is running on. Substance look and feel are available for all the platforms.

 Note: Restart JMicroVision to refresh the interface correctly.

 

Localization

The Regional Format selection changes the number and the date format in JMicroVision.

 Note: Use the regional format when entering or editing a number or a date in JMicroVision. For instance, the decimal point is a comma in some countries.
