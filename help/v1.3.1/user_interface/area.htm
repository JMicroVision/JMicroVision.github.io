Area Editor

This Area Editor allows defining specific areas for limiting the processing to a part of the image. The profile areas, which have a rectangular shape crossed by a dotted line that indicates the direction of the profile, are intended only for the Profile tool. The other areas, which are either rectangular, elliptic or polygonal, can be used by Data Viewer, Object Extraction, Point Counting or Image Factory.

For creating a new area, press one of the shape button and then click and drag the shape on the main image (a cross-cursor is displayed before and during the drawing time).

Note: The selected areas in the list are those visible on the image. For resizing or moving an area, select it and move the matching handle once the cursor has changed.

Rotate: Allows to rotate a profile area.

2D Drawing: Creates 2D drawings from the selected areas. Using this option gives information on the area position and the area measurements.

Note: When closing the dialog, all the rectangular areas that have a region outside the image are corrected so that they remain inside the image. For the other area types, a message asks the operator to correct them.

Note: The length and the width of an area must be larger than 4 pixels, otherwise the area is not saved. There is no restriction for the profile areas, the smallest width can be 1 pixel.
