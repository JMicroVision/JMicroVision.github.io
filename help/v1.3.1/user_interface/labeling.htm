Labeling

This dialog enables to attach a label on the vector drawings of the selected Tool in the Data Viewer.

Label: Select a parameter to display as a label.

Text max. (char.): This is the limit of characters that can be displayed for a text parameter (e.g. Name of Note).

Color: Select a font color.

Font: Customize the font size and type.

 Note: To open this window: In the Data Viewer , click on the Labeling button.
