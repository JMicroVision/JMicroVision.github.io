Shortcuts

Shortcut

Action
Arrows Move the image (when the hand is selected)
Move the selected drawings
Delete Delete the selected drawings or an item of a list
Double left click Show properties of a drawing or item of a list
Ctrl + A Select all (drawings, table elements or list elements)
Ctrl + C Copy table elements
Ctrl + left click Resize 3d charts
 mouse wheel Zoom in or zoom out any image view
Right click Open the context menu for the thumbnail, the drawings and the image views
Shift + left click Rotate 3d charts
Add or remove a specific drawing to the selection
H Hold down: activate the hand for moving the image
Key released: quit this mode
