The Windows, Solaris and Linux installers include a copy of the Java Runtime Environment (JRE*) and JAI* libraries (hence their large size), which will be installed in subdirectories of the JMicroVision installation. Thus, they will not conflict with or overwrite any existing JRE you may already have on your computer. The inclusion of the JRE ensures the connection between JMicroVision and the Java Runtime.
Read carefully the Installation instructions related to a specific platform!

*JRE: Java Runtime Environment   JAI: Java Advanced Imaging
